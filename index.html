About Me

저는 어려서부터 컴퓨터와 수학에 많은 관심을 가졌습니다. 특히 초등학생 때부터 집에 있는 Basic과 C/C++ 책에 관심이 생겨 시간날때마다 집에 있는 286XT 컴퓨터로 간단한 프로그램들을 만들어 보기 시작했습니다. 프로그램들을 만들고 실행했을때 오류가 발생할때 해결하고 로직을 새로 추가하고 문제없이 실행되었을때 희열을 느끼면서 현재 뒤돌아보면 이때부터 프로그래머로써 꿈을 가지게 되었던 것 같습니다.


대학교 재학시 학과 학생 대상으로 C/C++ 교육하는 소모임을 만들어 활동했으며, 인터넷 보편화를 통해 넷스케이프를 사용하여 인터넷을 사용하면서 웹 프로그래밍에 관심을 가지게 되었습니다. 이후 대학원에서 대학생 대상으로 APM(Apache-PHP-MySQL) 환경에서 PHP와 MySQL 교육을 진행했으며, 주변에 프로그래밍이 필요한 곳에 아르바이트로 응용프로그래머와 웹프로그래머로써 제 역량을 키워갔습니다. 시간이 남을 때는 자료를 공유하고 커뮤니케이션을 할 수 있는 게시판을 개발을하여 아는 지인들 끼리 사용을 했습니다.


대학원을 졸업하여 4년동안 전문병역특례로 일하면서 C/C++로 윈도우 기반 프로그래밍을 했으며, 이때 부터 디자인패턴에도 관심을 가지게 되고, 확장 라이브러리 STL와 Boost를 사용했으며, 윈도우 UI를 위해 MFC를 윈도우 ActiveX를 위해 ATL 프로그래밍을 사용하게 되었습니다. 이때 부터 N Tier 구조를 가지는 네트워크 기반 Client / Server 프로그램을 개발하기 시작했습니다. 또, MS에서 만든 언어인 닷넷 C#에 관심을 가지게 됩니다.


전문병역특례를 끝나고 지인들과 함께 팀을 이루고 네트워크 모니터링 툴을 만들게 됩니다. 이때 닷넷 프레임워크의 WPF, WCF 등이 나오면서 제품에 개발에 사용하게됩니다. UI단은 WPF로 개발, 서버단은 Java로 개발하는 경험을 하게 됩니다.


저는 사용자 중심 UI를 집중적으로 개발하고 싶었습니다. 데스크탐이든 웹브라우저이든 상관없이 개발에 관심이 많았습니다. 이때 이직한 회사가 ZUI(Zoomable UI) 기술을 중시하는 개발 회사였습니다. 여기서 jQuery, ActiveX, Silverlight, WPF 등 각각 다른 플랫폼에 동작하는 똑같은 타일 기반 ZUI를 가지는 제품을 개발 및 리드를 하게 됩니다.


회사에서 ZUI에 CCTV 영상을 표출하는 기능이 있으면 좋겠다는 아이디어로 영상 표출에 대한 관심을 가지게 됩니다. 이때만해도 대부분 네트워크 기반이 아닌 CCTV 솔루션이 대세였습니다. 많은 고난을 통해서 컨텐츠를 제작하는 에디터도 개발하고 여러개의 영상을 표출하는 솔루션을 만들게 됩니다.


저는 프로그래머의 꿈인 실리콘 밸리에 대한 환상이 항상있었습니다. 회사가 미국지사를 설립하면서 실리콘 밸리에 주재원으로 일하는 기회가 생기게 됩니다. 실리콘 밸리에서 저는 많은 외국 개발자들을 만나면서 빅데이터를 접하게 됩니다. 때마침 IoT를 접목한 빅데이터를 이용하는 C/S와 Web 솔루션 개발 리더로써 참여하게 되면서 많은 것들을 알게됩니다. 이때 Hadoop, Cassandra, Neo4J, MongoDB, Spark, Docker, Mono, NodeJS, React 등을 많은 플랫폼 공부하며 사용하게 됩니다.


실리콘 밸리에 주재원으로써 일정을 마무리하여 한국에 들오고 나서, 몇몇 지인들과 스타트업을 시작했습니다. 그리고 지금까지 새로운 프로그래밍 언어, 새로운 개발 방식, CI 환경 구축 새로운 기술에 관심을 가지고 고객이 만족하는 시각화 관련 여러가지 솔루션을 개발 및 리드를 하고 있습니다. 현재는 Azure를 이용한 CI 환경 구축, Docker를 이용한 쉬운 서비스 배포, Java를 이용한 Back-End단의 Spring Boot, NodeJS를 이용한 Front-End단의 React 혹은 Flutter, Python을 이용한 Deep Learning에 대해 연구하며 솔루션 개발에 접목하고 있습니다.


나이가 많아도 코딩은 C#(WPF), C/C++ 등을 메인 프로그래밍 언어로 실전에 문제 없이 사용할 수 있으며 Java(Spring Boot), Dart(Fultter), Python, Javascript(Node, React)은 속도가 느려도 사용할 수 있는 올그라운드 풀스택 개발자입니다. 또한 영어로 회화 가능하며 많은 경험을 통해 소프트웨어 및 서비스 설계, 배포 전략 및 실행해 봤습니다.
